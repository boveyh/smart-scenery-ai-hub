【利用規約】

■できること（OK）
・動画投稿（youtube、ニコニコ動画等）
・人格を入れること
・本モデルを用いた動画の収益化
・モデルの改変

■禁止事項（NG）
・政治、宗教、公序良俗に反する利用
・自作発言（クレジット表記は任意）
・モデルの再配布

【モデルの仕様】
Vroid Studio での仕様を推奨します。
PC版VTS：左手をカメラに認識させるとモデルが手を上げます。
　　　左手をグーパーさせると、モデルが手を振ります。
ios/android版VTS：舌を出すとモデルも舌を出します。
