# 云岭慢游 · 景区导览前端风格规范

## 通用样式规范

| 属性 | 值 | 说明 |
|------|-----|------|
| 主背景色 | `#F7F2E6` | 暖米浅卡其，所有容器/卡片底色 |
| 文字主色 | `#3D2C2A` | 深褐暖黑，主标题/正文 |
| 文字辅色 | `#8B6E57` | 棕褐色，标签/小标题/辅助文字 |
| 文字浅色 | `rgba(61,44,42,0.4~0.55)` | 说明文字/弱化信息 |
| 卡片背景 | `rgba(255,255,255,0.5~0.85)` | 半透明白，叠加暖底 |
| 边框/点缀 | `rgba(180,136,100,0.08~0.15)` | 浅棕边界 |
| 深色主按钮 | `#3D2C2A` / 文字 `#F7F2E6` | 实心填充 |
| 次要按钮 | 透明底 + `rgba(139,110,87,0.2)` 边框 | 线框样式 |
| 内嵌文字按钮 | 无框，色值 `#8B6E57` | 纯文字交互 |
| 页面外背景 | `#EDE4D3` | 主容器外部底色 |
| 大圆角 | `18px ~ 32px` | 容器/卡片/按钮统一大圆角 |
| 阴影 | 柔和浅阴影，如 `0 8px 40px rgba(61,44,42,0.10)` | 轻商务风格 |
| 字体 | `'Noto Serif SC','Songti SC','SimSun',serif` | 衬线体主字体 |
| 无衬线字体 | `'Noto Sans SC','PingFang SC',sans-serif` | 按钮/标签等辅助文字 |
| 分割方式 | 内边距 + 区块间距，无生硬分割线 | 靠留白区分区域 |

## 文字层级

| 层级 | 字号 | 字重 | 色值 | 用途 |
|------|------|------|------|------|
| L0 | 1.75rem | 700 | `#3D2C2A` | 主标题（左栏 h1） |
| L1 | 1.2rem | 700 | `#3D2C2A` | 右栏标题（h2） |
| L2 | 0.85rem | 600 | `#8B6E57` | 模块小标题 / 标签 |
| L3 | 0.85rem | 400 | `rgba(61,44,42,0.55)` | 说明正文 |
| L4 | 0.75rem | 400 | `rgba(61,44,42,0.5)` | 辅助说明/建议文字 |
| L5 | 0.65~0.7rem | 400 | `rgba(61,44,42,0.4)` | 最弱化信息 |

## 交互按钮分类

1. **实心填充主按钮** — `background:#3D2C2A; color:#F7F2E6; border-radius:24px`
   - 用于"顾问已接入"等主要 CTA

2. **线框次要按钮** — `background:transparent; border:1px solid rgba(139,110,87,0.2); color:#8B6E57`
   - 用于"推荐半日游""在线讲解"

3. **内嵌文字小按钮** — `background:none; border:none; color:#8B6E57; font-size:0.65rem`
   - 用于景点卡片内的"讲解xxx"
