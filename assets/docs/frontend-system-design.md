# 景区导览前端系统设计（test_frontend 风格版）

> 设计基调：暖米国风 · 大圆角卡片化 · 柔和浅阴影 · 左右双栏布局
> 色值体系：主背景 `#F7F2E6` / 页面外 `#EDE4D3` / 文字 `#3D2C2A` / 辅色 `#8B6E57`
> 实现方式：基于 test_frontend/index.html 的 CSS 体系 + 原生 JS/TS，无第三方 UI 库

---

## 一、系统概述

基于 10 个景区租户的多租户架构，构建一套完整的景区导览前端系统。每个景区拥有独立的 POI 列表、实时信息、数字人导览员、路线推荐能力。前端以 `test_frontend` 的暖米国风卡片化风格统一呈现。

### 1.1 技术栈

| 层 | 技术 |
|------|------|
| 构建 | Vite 5 |
| 框架 | React 18 + TypeScript 5 |
| 样式 | CSS 变量设计系统（零第三方 UI 库） |
| 状态 | 组件内 `useState` + 顶层租户上下文 |
| 路由 | `useState` 手工切换（页面少，无需 react-router） |
| 网络 | 原生 fetch + WebSocket，无额外依赖 |

```
│  │                             │    │  │  ●──●──○──○    │  │
│  │    ░░░ Live2D ░░░          │    │  │  古街→戏台→... │  │
│  │    ░░░ 数字人 ░░░          │    │  └──────────────────┘  │
│  │    ░░░ 形象 ░░░            │    │  ┌─ 景点卡片 2×2 ──┐  │
│  │                             │    │  │ ┌────┐ ┌────┐  │  │
│  │  [建立整体印象]             │    │  │ │POI1│ │POI2│  │  │

│  │  [古街牌坊] [临江戏台]    │    │  │ │POI3│ │POI4│  │  │
│  │  [城西茶楼]               │    │  │ └────┘ └────┘  │  │
│  └────────────────────────────┘    │  └──────────────────┘  │
│                                     │  ┌─ 路线建议 ──────┐  │
│                                     │  │  推荐从...出发   │  │
│                                     │  └──────────────────┘  │
│                                     │  ┌─ 操作按钮 ──────┐  │
│                                     │  │[顾问已接入] [荐]│  │
│                                     │  └──────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

**关键交互**：
- 用户说 → NDJSON 流式响应 → `text_chunk` 显示字幕 + `audio_url` 播放语音
- 语音播放时 Live2D 口型自动同步（`setMouthOpen`/`setMouthForm`）
- 右栏 POI 卡片、路线数据从 `GET /api/v1/pois` 动态获取

**数字人模型映射**：

| 租户 | 数字人 | Live2D 模型 |
|------|--------|------------|
| 西湖景区 | 西子 | Hiyori（优雅风格） |
| 灵山胜境 | 灵善 | OsageGirl（庄重风格） |
| 默认 | 小灵 | Haru（通用） |

## 五、API 数据流映射

| 前端操作 | API 调用 | 数据用途 |
|---------|---------|---------|
| 切换景区 | 更新 `X-Tenant-Id` header | 全局 |
| 加载首页 | `GET /api/v1/info/realtime` + `GET /api/v1/pois` | 天气/公告/POI |
| 加载景点列表 | `GET /api/v1/pois?category=` | POI 列表 |
| 加载景点详情 | `GET /api/v1/pois/{poiId}` | POI 详情 |
| 路线推荐 | `POST /api/v1/route/recommend` | 路线结果 |
| 拍照识别 | `POST /api/v1/vision/recognize` | 识别结果 |
| 智能咨询 | WebSocket `/ws/chat` | 文字对话 |
| 数字人对话 | `POST /api/v1/digitalhuman/chat` (NDJSON) | 语音+字幕 |
| 拥挤度查询 | `GET /api/v1/crowdedness/latest/{poiId}` | 拥挤数据 |
| 加载离线 FAQ | `GET /api/v1/knowledge/offline` | 离线兜底 |
