# 智慧景区 AI 导览平台 · 演示讲稿

> 总时长约 8 分钟

---

| 时间 | 画面呈现 | 台词/旁白 |
|------|---------|----------|
| **00:00-00:25** | 开场动画：灵山大佛、九龙灌浴、灵山梵宫三景快切，淡入登录页「灵山胜境」| 大家好，我们展示的是「智慧景区 AI 导览平台」。项目以无锡灵山胜境为示例景区，面向游客提供智能导览、数字人讲解、路线规划和拍照识物服务，同时为景区管理方提供数据分析看板与内容配置能力。平台采用游客端与管理端双入口设计，一套系统兼顾服务与运营。 |
| **00:25-00:55** | 登录页，展示「游客入口 / 管理员入口」两个入口卡片，光标悬停切换 | 平台分为两类角色。左侧的游客入口解决「怎么玩、看什么、怎么问」的问题：游客可以在手机或景区大屏上直接获取导览服务。右侧的管理员入口解决「内容怎么管、服务怎么调、数据怎么看」的问题：运营人员可以在后台配置知识库、调整数字人形象、查看游客行为分析报告。 |
| **00:55-01:35** | 进入游客端首页，从上到下扫过：深色身份区（天气/温度/拥挤度）、游览建议浮窗、门票信息、5 大核心景点推荐卡片 | 进入游客端后，首页以大图卡片形式展示景区概览。顶部是实时天气、温度和拥挤度信息——数据来自高德天气 API，实时更新。中部是游览建议和门票信息，游客不用提前查大量攻略就能快速了解。下方是 5 大核心景点推荐——灵山大佛、九龙灌浴、灵山梵宫、五印坛城、祥符禅寺，每张卡片都配有景区实拍图和详细简介，点击即可直达详情页。 |
| **01:35-02:25** | 点击左侧「景点列表」，展示分类筛选栏（佛教文化/历史文化/自然风光），切换分类后点击「灵山大佛」卡片进入详情页 | 景点列表按分类展示全部 16 个景点点位，支持按佛教文化、历史文化、自然风光等分类筛选。每个景点卡片包含实景图、分类标签、拥挤度标识、游览时长和门票信息。点击「灵山大佛」进入详情页：左侧是景点实拍大图（支持三张轮播切换），右侧集中展示开放时间、建议停留时长、拥挤度进度条和详细的景点文化故事——比如大佛高 88 米、用铜 725 吨、216 级登云道等数据，方便游客现场决策。 |
| **02:25-03:30** | 进入「数字人导览」页面，展示左侧 Live2D 数字人画布（9 个角色可切换）、右侧聊天框，演示快捷提问和流式回答 | 项目的核心亮点是 AI 数字人导览。左侧是 Live2D 实时渲染的数字人，系统提供 9 个不同角色：活力满满的 Haru、温柔古典的 Hiyori、冷静理性的 871、神秘穿越的 Z、江南才女阮梅等，每个角色都有独立的人设提示词、TTS 语音和 Live2D 模型。游客可以直接在右侧聊天框提问，例如「灵山大佛有多高」或「九龙灌浴几点表演」。后端将回答按句切分，逐句生成语音并流式返回到前端；前端同步播放语音、更新聊天气泡、显示字幕，并通过实时音量分析驱动数字人的口型动作，让讲解更自然。回答过程中可随时点击「停止」中断播放。 |
| **03:30-04:10** | 演示点击快捷问题「介绍灵山胜境」，展示聊天气泡逐句追加、画布字幕显示、右上角"正在讲解"状态标记 | 这里可以看到，回答不是一次性等很久才全部出现，而是边生成边展示——每生成一个句子就立即显示在聊天气泡中，同时播放对应的语音。画布底部显示当前字幕，左上角有绿色的「正在讲解」状态标识。系统同时支持文字输入和图片上传（结合千问 VL 多模态模型），也支持停止当前讲解，适合真实景区里碎片化、低等待的交互场景。 |
| **04:10-05:00** | 进入「路线规划」页面，展示高德 3D 地图和右侧推荐路线卡片，点击「历史文化深度游」和「亲子家庭轻松游」查看路线差异 | 路线规划模块基于高德 3D 地图（支持俯仰角旋转），根据不同的游览目标推荐路线。系统内置了四条预设路线：历史文化深度游（6 个景点，步行约 2.5km）、自然风光全景游、亲子家庭轻松游、朝圣祈福专线。选择路线后，地图上会自动绘制贴合景区真实道路的路径——这些坐标数据是我们通过高德坐标拾取器手动采集的，不是简单的两点直线。地图上同时标注景点顺序编号，右侧面板显示总里程、预计时长和每段的分时信息。 |
| **05:00-05:40** | 在路线页点击地图上的景点标记弹出信息浮窗，点击右侧「全部景点」列表中的某个景点高亮定位 | 地图上每个景点都有圆角卡片标记，点击弹出信息窗，包含景点图片、简介和开放时间。右侧的「全部景点」列表也支持点击定位——点击某个点位后地图自动聚焦并弹出信息窗。这样从「想去哪里」到「怎么过去」形成完整闭环，减少游客在景区内迷路和反复查地图的成本。路线支持步行、驾车、公交三种出行方式，每种方式的预估时间和距离动态计算。 |
| **05:40-06:20** | 进入「拍照识物」页面，拖拽上传一张景区图片，选择快捷问题「这是什么景点？」，展示识别结果 | 拍照识物模块支持上传 JPG、PNG、WebP 格式的图片。游客在景区看到建筑、雕塑或植物时，可以先拍照，再选择系统提供的快捷问题或自定义提问。系统会调用通义千问 VL 多模态模型进行视觉识别，结合 RAG 知识库中该景点的历史文化和故事信息，返回详细的讲解结果。这样就把「看不懂」变成了「边走边学」，让游览过程更有深度。 |
| **06:20-06:55** | 进入「实时资讯」页面，展示左侧天气面板、景区公告、16 个景点拥挤度列表和错峰建议 | 实时资讯页面顶部展示实时天气（高德天气 API），包括温度、湿度、风向风力。左侧是景区公告栏——展示当日的表演时间调整、银杏观赏期提示、停车位状态等动态信息。右侧是 16 个景点的拥挤度列表，按拥挤程度降序排列，高峰点位红色标记。底部根据当前总体拥挤度动态显示错峰建议文案：人多时建议先去空闲点位，人少时建议按推荐路线依次游览。 |
| **06:55-07:35** | 切换登录回到登录页，选择「管理员入口」，进入管理端，展示侧边导航栏的四个功能入口 | 现在我们切到管理员端。登录后左侧导航栏显示四个功能入口：数据大屏概览、知识库管理、数字人配置和游客报告。管理端和游客端使用完全相同的暖米国风设计体系，保证视觉一致。 |
| **07:35-08:10** | 展示「数据大屏概览」页面：4 个 KPI 卡片（今日服务人次、本周服务人次、平均满意度、知识库条目）、近 7 日服务趋势柱状图、热门话题排行 | 数据大屏展示景区运营的核心指标。顶部四个数字卡片显示今日服务人次、本周服务人次、平均满意度和知识库条目数量。左侧的「近 7 日服务趋势」柱状图通过基线偏移算法放大数据间的差异，让趋势一目了然。右侧的「热门话题」排行展示游客最关心的问题——灵山大佛、九龙灌浴表演、门票价格等。下方是景点游览排行和游客年龄分布，帮助管理者判断哪些点位压力最大、不同年龄段的满意度差异。 |
| **08:10-08:45** | 展示「知识库管理」页面：知识条目列表和 FAQ 管理标签页，展示搜索和编辑功能 | 知识库管理页面支持管理员按租户管理知识条目和 FAQ。知识条目包含标题、内容、标签、关联 POI 和来源等字段，支持关键词搜索、新增、编辑和软删除。FAQ 管理支持常见问题的增删改查，每条 FAQ 会记录点击次数，帮助运营人员判断哪些问题是游客最常问的。后台数据同时为 RAG 知识检索提供素材支撑。 |
| **08:45-09:10** | 展示「数字人配置」页面：左侧配置列表、右侧 Live2D 预览画布、编辑弹窗 | 数字人配置页面采用左右分栏布局。左侧列出所有数字人配置（Haru、Hiyori、871、Z、阮梅等 9 个角色），点击任意一条，右侧画布立即渲染对应的 Live2D 模型实时预览。编辑弹窗支持修改人设名称、TTS 语音和语速音调、关联的 Live2D 模型，以及最重要的——人设提示词。提示词决定了 AI 以什么角色、什么语气、什么风格来回答游客，是整个交互体验的灵魂。 |
| **09:10-09:40** | 展示「游客报告」页面：满意度分布饼图、停留时长散点图、消费结构分析、服务建议 | 游客报告页面基于 14 万条真实行为数据（包含年龄、性别、停留时长、5 类消费金额等字段），展示满意度分布饼图、各年龄段满意度趋势线、停留时长 vs 满意度散点图、消费结构分析和服务建议。底部给出了数据驱动的三条业务结论：高门票和长停留是满意度的主要负面影响因素；中老年游客对门票价格敏感；建议主推 2 人短途套餐。 |
| **09:40-10:00** | 总结页面：项目核心能力回顾 | 以上就是「智慧景区 AI 导览平台」的全貌展示。项目从游客的游览体验出发，覆盖了导览、讲解、路线、识物、资讯五大场景，同时为管理方提供了数据驱动的运营决策支持。基于通义千问 VL 多模态模型、Live2D 数字人引擎和高德地图服务，打造了一套可落地、可扩展的智慧景区解决方案。感谢观看。 |

## 演示要点备忘

### 关键页面 URL
- 登录页：`http://localhost:5173/`
- 游客端 → 首页总览 / 数字人导览 / 景点列表 / 路线规划 / 拍照识物 / 实时资讯
- 管理端 → 数据大屏 / 知识库管理 / 数字人配置 / 游客报告

### 数字人切换
9 个角色：Haru（活力）、Hiyori（古典）、871（理性）、Z（神秘）、阮梅（才女）、BetaSmodel（元气）、Kirin Kirinja（奇幻）、Osage Girl（和风）、Half-Demon Elf（灵性）

### 演示建议
- 第一问用「介绍一下灵山胜境」展示长回答流式输出
- 第二问用「九龙灌浴几点表演」展示单点直答
- 快速提问按钮可快速触发
- 停止按钮展示中断功能
- 路线页面展示两条不同路线对比（文化 vs 亲子）
- 管理端先展示数据大屏，再展示知识库或数字人配置
